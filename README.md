# 🤖 Level 2: Machine Learning Tasks

This repository contains my projects for Level 2, focusing on Classification algorithms using Scikit-Learn.

## 📊 Task 1: Customer Churn Prediction
- **Algorithm:** Logistic Regression.
- **Objective:** Predict whether a customer will leave the service based on usage patterns.
- **Key Concepts:** Data Scaling, Binary Classification, and Performance Metrics (Accuracy, Recall).

## 🌳 Task 2: Iris Flower Classification
- **Algorithm:** Decision Tree Classifier.
- **Objective:** Classify iris flowers into three species based on petal and sepal measurements.
- **Key Concepts:** Feature Encoding, Model Visualization, and Decision Boundaries.

## 🛠️ Tools Used:
- Python (Pandas, NumPy, Scikit-Learn, Matplotlib).
- Google Colab / VS Code.
